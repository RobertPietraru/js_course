Operatori de Comparare

În condițiile if putem folosi diferiți operatori de comparare:

Operator Descriere
== Egal (verifică doar valoarea)
=== Strict egal (verifică valoarea și tipul)
!= Diferit
!= Strict diferit
> Mai mare
< Mai mic
>= Mai mare sau egal
<= Mai mic sau egal
