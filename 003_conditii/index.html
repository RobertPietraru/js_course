Valori Truthy și Falsy

În JavaScript, orice valoare poate fi convertită la boolean. Valorile care se convertesc la true se numesc "truthy", iar cele care se convertesc la false se numesc "falsy".

Valori Falsy

Următoarele valori sunt întotdeauna falsy:

  • false
  • 0
  • "" (string gol)
  • null
  • undefined
  • NaN

Valori Truthy

Toate celelalte valori sunt truthy, inclusiv:

  • Orice număr diferit de 0 (inclusiv negative)
  • Orice string nevid
  • Obiectul {}
  • Array-ul []
