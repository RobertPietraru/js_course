Tipuri de Date Avansate în JavaScript

Array-uri (Tablouri)

Ce sunt Array-urile?

Array-urile sunt structuri de date care pot stoca multiple valori într-o singură variabilă. Ele sunt ordonate și pot conține orice tip de date.

Exemple de operații cu array-uri:

  • push() - adaugă un element la sfârșit
  • pop() - elimină ultimul element
  • shift() - elimină primul element
  • unshift() - adaugă un element la început
  • splice() - elimină sau înlocuiește elemente de la o poziție specificată
  • slice() - extrage o porțiune din array

null și undefined

JavaScript are două valori speciale pentru reprezentarea absenței unei valori:

undefined

undefined apare când:

  • O variabilă este declarată dar nu i s-a atribuit o valoare
  • Încercăm să accesăm o proprietate care nu există (vom vedea mai târziu)
  • O funcție nu returnează explicit o valoare (vom vedea mai târziu)
