Pasul 2: Instalarea Extensiei Live Server

Tutorial Video pentru Live Server:

  1. Deschideți VS Code
  2. Faceți clic pe iconița Extensions din bara laterală stângă (sau apăsați Ctrl+Shift+X)
  3. Căutați "Live Server" în bara de căutare a extensiilor
  4. Căutați extensia creată de Ritwick Dey
  5. Faceți clic pe "Install" și așteptați finalizarea instalării

Pasul 3: Utilizarea Live Server

  1. Deschideți folderul proiectului în VS Code
  2. Creați sau deschideți un fișier HTML
  3. Faceți clic dreapta în editor
  4. Selectați "Open with Live Server" din meniul contextual
  5. Browserul dvs. implicit se va deschide automat cu pagina dvs.

Caracteristici Principale ale Live Server

  • Reîncărcare automată a browserului la modificarea fișierelor
  • Suport pentru majoritatea browserelor
  • Configurare personalizată a portului
  • Suport HTTPS

Depanarea Live Server

Dacă Live Server nu funcționează:

  • Asigurați-vă că ați salvat fișierul HTML
  • Verificați dacă portul 5500 nu este utilizat de o altă aplicație
  • Încercați să reporniți VS Code
  • Verificați dacă antivirusul nu blochează conexiunea

Notă: Veți vedea un buton "Go Live" în colțul din dreapta jos al VS Code când Live Server este instalat corect
