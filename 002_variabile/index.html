Variabile în JavaScript

Ce sunt Variabilele?

Conceptul de Variabilă

O variabilă este un container pentru stocarea datelor. Gândește-te la ea ca la o cutie în care poți păstra diferite tipuri de informații.

În JavaScript, poți declara variabile folosind:

  • let - pentru variabile care pot fi modificate
  • const - pentru valori constante care nu se modifică
  • var - varianta mai veche, mai puțin recomandată în codul modern
